# Images Directory

Add your property photos here with the following suggested filenames:

## Required Images:
1. **cottage-exterior.jpg** - Main exterior view of the cottage
2. **exterior-1.jpg** - Additional exterior view
3. **living-room.jpg** - Living room interior
4. **bedroom.jpg** - Bedroom photo
5. **kitchen.jpg** - Kitchen area
6. **bathroom.jpg** - Bathroom
7. **garden.jpg** - Garden/outdoor space
8. **outdoor-seating.jpg** - Outdoor seating area
9. **view.jpg** - Scenic views from the property

## Image Guidelines:
- Use high-quality images (at least 1200px width)
- Optimize images for web (use tools like TinyPNG or ImageOptim)
- Supported formats: JPG, PNG, WebP
- Keep individual file sizes under 500KB for faster loading
