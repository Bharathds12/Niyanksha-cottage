## 🔄 How to Update Rules

### To Modify a Rule:
1. Open `index.html`
2. Find the House Rules section (around line 236)
3. Locate the specific rule card
4. Edit the text
5. Save and push to GitHub

### To Add a New Rule:
1. Copy an existing rule card HTML block
2. Change the icon emoji
3. Update the heading
4. Update the description
5. Save and push to GitHub

### To Mark as Important:
Add `important` class to the rule-card:
```html
<div class="rule-card important">
```
